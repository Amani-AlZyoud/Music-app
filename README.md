# Music-app
Tunes aims to provide a comprehensive platform for music enthusiasts, offering a seamless experience of exploring music and discovering insightful articles.

## technologies used in this project:
- React.js
- Node.js
- Tailwindcss
- Redux
- Helmet
- MongoDB
- JSONwebtoken (JWT)
